# 🎮 Hangman Game

## 🔹 Description

A classic word guessing game where the player must guess the hidden word one letter at a time before running out of attempts.

## ✨ Features

* 🎲 Random word selection
* 🔁 Continuous gameplay loop
* ⚠ Input validation
* ❌ Limited attempts (6 chances)
* 🎉 Win/Loss detection

## 🧠 Concepts Used

* `random` module
* loops (`while`)
* conditionals (`if-else`)
* strings & lists

## ▶️ Run the Game

```bash
python hangman.py
```
